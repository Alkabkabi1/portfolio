## 🚀 Run locally

```bash
git clone https://github.com/Alkabkabi1/portfolio.git
cd portfolio
```

Open `index.html`, or:

```bash
python -m http.server 8080
# → http://localhost:8080
```

Switch to **English** with the locale button, or preset in DevTools:

```js
localStorage.setItem('portfolio-locale', 'en');
location.reload();
```
